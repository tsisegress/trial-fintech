# trial-fintech

Frontend prototype for the Fintech AI matching experience.

## Run locally

```bash
npm install
npm run dev
```

## Logo placement

Place your logo file at:

```text
public/branding/fi.png
```

`BrandMark` will automatically try this image first and fall back to the `F` mark if missing.

## Tests

```bash
npm test
```
